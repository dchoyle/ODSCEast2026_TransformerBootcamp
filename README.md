# ODSCEast2026_TransformerBootcamp
Material for the ODSC AI East 2026 introductory tutorial on the mathematics of transformers and LLMs

# Prerequisites for ODSC AI East 2026 Bootcamp: Introduction to math behind Transformers and LLMs
## Technical prerequisites:
A full Python installation, including the following packages
*	NumPy (>= 2.4.0)
*	Matplotlib (>= 3.10.8)
*	gensim (>= 4.4.0)
*	PyTorch (>= 2.10.0)
  
and the ability to run Jupyter notebooks. I would recommend a full Anaconda installation (downloadable from www.anaconda.com/download) and installing the additional gensim and PyTorch packages.

## Mathematical prerequisites:
*	A knowledge of basic probability concepts, including conditional probability
*	A knowledge of basic linear algebra concepts, including vectors, matrices, inner products between vectors, matrix multiplications, and matrices as representations of linear transformations
*	A knowledge of basic machine learning concepts, including the idea of training of model parameters through minimization of a loss function.
